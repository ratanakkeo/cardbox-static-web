== Description ==

Legacy Book Club is a sophisticated WordPress theme crafted specifically for book-related websites. With its clean typography and thoughtful layout, it's perfect for book clubs, reading groups, literary blogs, and book review websites.

Major features include:

* Custom banner styling with elegant typography
* Dedicated sections for book reviews and discussions
* Special offer section with styled headings
* Mobile-responsive design
* Custom font integration including Pacifico for stylistic elements
* Clean, reader-friendly layout

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button
2. Click on 'Upload Theme' and choose the theme's .zip file
3. Click on 'Install Now' and then 'Activate' to use your new theme
